# 01ext

a expansion board with esp-01/esp-01s.

![pcb](https://github.com/bj5/01ext/blob/main/docs/01ext-3d.png)


01ext  come with
- usb-serail chip 
- 4 pin serial io connect
- a switch to  flash esp-01/01s
